妳茫然看著鏡中的自己



<!--more-->



儘管妳從不停止堆砌各種保養品，仍掩蓋不住歲月的刷洗



妳的工作是取悅男人 — 妳做的很好，他們說妳是店裡最優秀的 — 但那不是妳的目的，起碼現在不是，不完全是



開始打扮吧



Chanel的香水 — 擦的是大麻的味道
\
Dior的唇膏 — 塗的是霓虹燈的顏色
\
Carolina Herrera的真絲襯衫 — 裸露的是性感的慾望，包覆的是骯髒的肉體
\
Ralph Lauren的皮革大衣 — 左邊的口袋塞滿的是鈔票
\
右邊的口袋塞滿的是垃圾



今晚妳喝的很醉
\
男人說妳依舊美麗，甚至勝於當年
\
他們送妳量身訂製的禮服、精密切割的鑽石
\
卻看不見妳長裙下微跛的步伐
\
妳忽然想起左腳的傷疤
\
25年了，癒合的傷口至今仍隱隱作痛



深夜，妳來不及脫去衣服便倒臥在床
\
這座城市的燈火總是通明，但妳仍沉沉睡去



夢裡，妳想起了妳的祖先
\
他們在400年前搭著小船迎著巨浪，在黑暗的大洋中航行
\
他們不知道眼前有什麼，但仍不畏險阻勇往直前
\
妳被他們的精神所動，眼角不住流下淚水



天亮了，起身的妳仍向著那夢中的新大陸前進

初次造訪紐約，給我的印象是矛盾



隨處可見的警力與犯罪猖狂的地鐵
\
寬敞筆直的大道與壅塞停滯的交通
\
遼闊無際的森林與狹窄擁擠的公寓
\
珠光寶氣的貴婦與無家可歸的遊民



它融合了太多的文化與太深的歷史，不同力量拉扯之下處在一個絕妙的平衡點上



創作這首詩時，我腦中想的《大亨小傳》。當Gatsby望著對岸的綠色燈火時，他知道他所期盼的是如此接近 — 然而事實上那是如此遙不可及。



Gatsby如此，紐約如此，北漂的我們如此。
