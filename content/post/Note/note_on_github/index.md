Check out my notes on Github: [Link](https://github.com/kengyuchen/Notes)

I have started a project to gradually move my notes to Github. I have accumulated $\aleph_0$ notes since I was an undergraduate student, including mathematics, cryptography, programming, and some random thoughts. I have been using Notion to manage them for a while, but I find it hard to organize them in a way that is easy to navigate and share. Rewriting them in Latex and moving to Github will allow me to have more control over the formatting and structure of my notes, and also make it easier for others to access and contribute to them. I will try to keep the notes updated and organized, and I hope they can be useful for others who are interested in similar topics.

For now, only mathematics and cryptography notes are available. I will gradually add more notes on programming and other topics in the future. If you have any suggestions or feedback, please feel free to contact me.
